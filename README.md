# 顶点小说 Flutter版本
使用flutter写的一款小说阅读软件
## 使用技术
getx做状态管理和路由管理
dio进行网络请求
hive进行简单数据存储
sqflite进行图书内容存储
## 已完成功能
基本功能开发,包括图书搜索,详情查看,图书阅读,书架管理等
